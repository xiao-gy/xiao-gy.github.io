萌新也能看懂的黑苹果教程

Posted by xiao-gy on 2020-04-04
xiao-gy's blog
  • 主页

这一切都要从一只蝙蝠说起

故事开始：

从前，有一个人，他真的很闲，喜欢玩电脑，喜欢乱搞，但是他乱搞的时间不多，直到一只蝙蝠的出现，改变了他的命运。

废话不多说，进入正题

准备工作：

  • 一个黑苹果镜像，推荐去黑果小兵那里找，最新的或者Mojave都可以

  • 一个镜像烧写工具，推荐全平台通用的小白工具balena Etcher

  • 一个分区工具，推荐diskgenius

  • (非必须)一个远景论坛账号

  • 一颗爱折腾的心

  • 如果你下载的镜像有标注(双UEFI)的字样，那就可以省略，否则请准备一个U盘PE，推荐用微PE工具箱，请在安装前确定一下你的PE可以正常使用 (鬼知道因为少了这一步我折腾了一天)

    以上就是需要准备的全部工具

开始进行win下的操作

1.打开你的分区工具，这里以diskgenius为例。先划出一块空间，这样你就得到了一块空白的空间，注意，这块空间少说要有50GB左右！

2.新建分区，右键那块空白的空间，选择新建分区，在格式里面找到APFS，按下确定，然后按下软件右上角的保存分区表来保存修改。

以上这几步因为简单，所以我就不加配图了 (其实是因为我懒得去截图了) ，不会的可以自己去百度diskgenius的使用方法

3.写入镜像，打开你的balena Etcher，你大概会看到三个按键(如图)，第一个选你下载的系统镜像，第二个选择你的U盘，然后按下最后的按键

Flash！

等待到出现如下界面就可以拔出U盘了

4.U盘启动电脑(如果启动失败的话，请自行百度 如何关闭安全启动 )

开始在mac安装盘下的操作

首先是一些问题，如果下载的是Catalina版的镜像，在跑代码之前卡在一排加号++++++++++++++++++++的话，两个解决方法：

1.有自己加过内存条的尝试拔掉新加的内存条
2.考虑使用Mojave的镜像(本人就是这样解决的233)

1.假定你成功进入了安装盘，首先进入磁盘工具，抹掉你要安装mac的磁盘，格式选择APFS。

这里有一个很玄学的操作，如果你无法抹掉你的那个硬盘的话，请在确保你的EFI启动分区大于200MB的情况下观赏博主的玄学解决法：

1.找一个U盘，在mac安装盘下抹掉，格式为`APFS`，如果可以的话进行下一步
2.选择你要安装mac的盘，按下恢复，在源磁盘选择你的U盘，然后恢复。
3.如果成功的话就正常安装，如果失败的话，尝试再次抹掉你你要安装mac的磁盘为`APFS`

2.如果你安装的是Mojave的话，请在安装之前先打开终端，输入以下命令

3.进入安装程序，同意所有的条款，在选择磁盘的界面选择你的事先抹掉的磁盘，接下来的操作都很人性化了

进行第二阶段的安装

在安装的过程中会重启，如果你错过了也没关系，在开机的时候进入Clover的界面，选择install mac from [你的磁盘名]，如果没有命名的话，磁盘名应该为未命名。

然后进入第二阶段之后就等待就好了

进入系统

排除那些被卡在五国图的情况，那么你应该进入系统了。

如果你被卡在五国图，那么请自己去远景论坛查找解决方法。

在开机之后，先测试一下你的WiFi、扬声器是否正常，如果不正常的话，进行驱动的修复。

首先是WiFi的修复，在win下右键此电脑(或其他什么名字) - 管理 - 设备管理器 - 网络设备(忘了叫什么) - 找到你的WiFi网卡型号 - 上github搜索 - 自己百度安装去

接着是显卡的修复，首先下载Clover的配置软件(在mac下运行)

链接: https://pan.baidu.com/s/1vu-RIW9H9Sk41tzKhl2-ZA 提取码: bj1m

先挂载你U盘的启动分区

打开你的配置文件

查看你的显卡，这里以独显为例

可以看到我的显卡是Intel HD Graphics 4400，然后记下来

来这里找的你的显卡对应的platform-id，然后在第二个框框选择你的显卡，如果后面有跟着mac的型号，需要在过来这里修改你的mac的型号。

然后重启电脑，再打开关于本机，如果显存被正确识别了，就可以了。

接下来就是声卡的修复了。

首先，在我发出的百度云链接里有安装的程序，下载下来，打开。

先选择下面的一二项，再将VoodooHDA-2.9.2.kext拖入上面的框框，再按下执行，时间可能有点久，等会就好了。

基本上就安装成功了，如果还有什么需要请自己去找驱动安装
